# Exercise 1: Control Structures

## Objective

The objective of this exercise is to understand and implement PL/SQL control structures such as loops and conditional statements to perform various banking operations.

## Scenarios Covered

* Apply a **1% interest rate discount** to customers above 60 years of age.
* Promote customers with a **balance greater than 10,000** to VIP status.
* Display reminder messages for **loans due within the next 30 days**.

## Concepts Used

* PL/SQL Anonymous Blocks
* FOR Loop
* IF-THEN Statement
* UPDATE Statement
* Date Functions (`SYSDATE`, `MONTHS_BETWEEN`)
* `DBMS_OUTPUT.PUT_LINE`
* Transaction Control (`COMMIT`)

## Files

* `Exercise1.sql` – Contains solutions for all three scenarios.
* `Ex1.md` – Documentation for Exercise 1.

## Learning Outcomes

After completing this exercise, I learned how to:

* Iterate through records using PL/SQL loops.
* Apply conditional logic using IF statements.
* Update database records based on business rules.
* Work with Oracle date functions.
* Display execution messages using `DBMS_OUTPUT`.
* Commit database transactions after successful updates.
