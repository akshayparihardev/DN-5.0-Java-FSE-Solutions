## Overview

This repository contains my completed hands-on exercises for the **Cognizant Digital Nurture 5.0 Java Full Stack Engineer** learning journey. It is organized week by week and covers core Java foundations, design patterns, data structures, SQL, testing, Spring, REST services, microservices, and React frontend development.

The goal of this root README is to provide a clean, review-friendly entry point for mentors, reviewers, and team leads. Individual exercises contain their own notes, source code, and implementation details inside the respective module folders.

## How to Review This Repository

1. Start with the week-wise map above to understand the overall coverage.
2. Open the topic folder for the area you want to review.
3. For Java, Spring, SQL, and Git exercises, check the individual `Ex*.md`, `HOL*.md`, or `Handson*.md` files along with the source files.
4. For React exercises, open each application folder and review its local `README.md`, `src/`, and `package.json`.
5. Use the setup commands below to run selected projects locally.

## Prerequisites

Install the following tools before running the projects:

| Tool | Purpose |
| --- | --- |
| JDK 17 or compatible JDK | Java and Spring projects |
| Maven | Build and test Maven projects |
| Node.js and npm | React applications |
| Git | Repository cloning and version control |
| SQL database client | PL/SQL and database exercises, where applicable |
| IDE | IntelliJ IDEA, Eclipse, VS Code, or Spring Tool Suite |

## Getting Started

Clone the repository:

```bash
git clone https://github.com/akshayparihardev/DN-5.0-Java-FSE-Solutions.git
cd DN-5.0-Java-FSE-Solutions
```

Run a Maven project:

```bash
cd WEEK_3/Spring_REST/spring-learn
mvn clean test
mvn spring-boot:run
```

Run a React application:

```bash
cd WEEK_5/React/Handson_1/myfirstreact
npm install
npm start
```

Compile a standalone Java exercise:

```bash
cd WEEK_1/DSA/Exercise_1_Inventory_Management_System
javac *.java
java Main
```
